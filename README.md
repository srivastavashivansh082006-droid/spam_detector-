
## 📨 AI Spam Mail Detection Using NLP Classification: A Machine Learning-based Text Classifier for Differentiating Between Spam and Ham (Legitimate) Mails.

## 👨‍💻 Digital Literacy (CSE0001) Course at VIT Bhopal
This project demonstrates how Artificial Intelligence can provide practical applications that can help enhance digital security and filter communication.

## 📋 Data Pipeline Development & Implementation
As an aspiring data analyst, I created a solid data pipeline to perform text preprocessing, feature engineering, and model evaluation to ensure high precision in identifying malicious threats to users.

## 💻 Technology Used
* **Programming Language: Python 3.x**
* **Libraries: Pandas - Data Manipulation & Analysis, Scikit-Learn - Machine Learning Modeling (Naive Bayes & Random Forest), Matplotlib & Seaborn - Data Visualization & Performance Metrics, NLTK/String (Natural Language Toolkit) - Text Preprocessing & Cleaning.**
* **Working Environment: Visual Studio Code.**

## 🛠 Main Features of This Project
* **Advanced Text Cleaning** - Automation of punctuation, stop-word removal, and case normalization for maximizing model signal.
* **TF-IDF Vectorization** - Used Term Frequency-Inverse Document Frequency (TF-IDF) to determine the relative importance of selected "spammy" keywords.
* **Multiple Model Comparison** - Comparison of several algorithms (Multinomial Naive Bayes vs Random Forest) in order to select the most efficient model.
* **Confidence Score** - In addition to predictions, the user will also receive a Confidence Percentage with each prediction.
